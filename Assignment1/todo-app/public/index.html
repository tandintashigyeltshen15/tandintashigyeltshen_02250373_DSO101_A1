To-Do App Deployment Assignment

1. Project Overview

This assignment involves creating a simple To-Do Web App with frontend, backend, and database, then deploying it using Docker and Render. It also includes automating deployment through GitHub.

  • Frontend: React UI with Add, Edit, Delete tasks functionality
  • Backend: Node.js + Express API with CRUD operations
  • Database: PostgreSQL on Render
  • Deployment: Docker images pushed to Docker Hub, deployed on Render

2. Database Setup

Created PostgreSQL database on Render:

  • Service Name: todo-db
  • Database Name: tododb_abc123 (auto-generated)
  • Credentials: Use DATABASE_URL for connection

Environment Variables for backend:

DATABASE_URL=postgres://user:password@dpg-abc123.render.com:5432/tododb_abc123
PORT=5000

Note: Do not push .env file to GitHub.

3. Backend Setup

Connected Node.js backend to PostgreSQL using pg library:

const { Pool } = require('pg');

const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
});

module.exports = pool;

APIs implemented:

  • GET /tasks
  • POST /tasks
  • PUT /tasks/:id
  • DELETE /tasks/:id

4. Dockerization

Backend Dockerfile

FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 5000
CMD ["node", "server.js"]

Frontend Dockerfile

FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npx", "serve", "-s", "build"]

Docker commands used:

# Build backend
docker build -t tandintashigyeltshen/be-todo:02250373 .

# Push backend
docker push tandintashigyeltshen/be-todo:02250373

# Build frontend
docker build -t tandintashigyeltshen/fe-todo:02250373 .

# Push frontend
docker push tandintashigyeltshen/fe-todo:02250373

5. Render Deployment

Backend Service:

  • Image: tandintashigyeltshen/be-todo:02250373
  • Environment Variables: DATABASE_URL, PORT

Frontend Service:

  • Image: tandintashigyeltshen/fe-todo:02250373
  • Environment Variable: REACT_APP_API_URL=https://be-todo.onrender.com

6. Automation (render.yaml)

services:
  - type: web
    name: be-todo
    env: docker
    dockerfilePath: ./backend/Dockerfile
    envVars:
      - key: DATABASE_URL
        value: postgres://user:password@dpg-abc123.render.com:5432/tododb_abc123
      - key: PORT
        value: 5000

  - type: web
    name: fe-todo
    env: docker
    dockerfilePath: ./frontend/Dockerfile
    envVars:
      - key: REACT_APP_API_URL
        value: https://be-todo.onrender.com

Push to GitHub → Connect repo to Render → Automatic deployment occurs on each commit.
