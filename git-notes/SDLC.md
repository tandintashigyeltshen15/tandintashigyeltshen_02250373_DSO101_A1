## SDLC - Software Development Life Cycle

===============================================================================================

# There are 4 models for SDLC:
1. Waterfall model - One of the best model before
2. Spider model
3. Ajile model
4. V-Shaped model

# 6 phases in waterfall model
1. Planning
2. Requirement Analysis
3. Designing
4. Implementation
5. Testing
6. Deployment and Maintainance

## What was the biggest issue for this model?

> This model is very rigid.
> This model isnt flexible. It cannot move back to its initial phase.

## Which model overcame this issue?

> A new methodology was developed called AJILE Method.
> This method is flexible.
> It can change its feature any time.

# DevOps - Subset of SDLC

> It focuses on CICD (Continuous Integration and Continuous Deployment)
